# Aera---Square

##
Aim

Write a Program to calculate the area of a Square

###
code
```

#include<stdio.h>
int main()
{
    float l, a;
     
     scanf("%f", &l);
     
     a = l*l;
     printf("%f", a);
    return 0;
}
```

##
Output: Thus the area of the square got printed to the output terminal.
